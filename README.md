# Vehicle Inventory System

## Overview

This project is a Java-based vehicle inventory management system developed as part of an object-oriented programming assignment.

## Features

- Vehicle management using Java classes
- Customer management
- Sports car specialization through inheritance
- Dealership inventory system
- Object-oriented programming principles
- Java testing using ProjectOneTester

## Technologies

- Java
- Object-Oriented Programming (OOP)

## Files

- Vehicle.java
- SportsCar.java
- Customer.java
- Dealership.java
- ProjectOneTester.java
